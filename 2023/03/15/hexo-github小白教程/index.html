## 搭建github仓库
### 创建仓库
1. 申请一个新的代码仓库用于保存网页
- 点击 your repositories,进入仓库页面
- 点击New按钮，进入仓库创建页面

  • 填写仓库名，格式必须为<用户名>.github.io，然后点击Create repository
  • 点击creating a new file创建一个新文件，作为我们网站的主页
  • GitHub Pages中找到我们主页的地址为 https://starry407.github.io/
